Built for https://tenfoldcars.co.uk/

When a user completes the form, a POST request is made to the API. Regestration of the vehicle is then input into the DVLA vehicle check system and properties of interest alongside the website form properties are emailed to the website owner.

Uses Vercel, EJS, and Resend.


Database editor
Lets the owner/partners make changes to the owned vehicle database without having to directly edit the Postgres database.
